Solution

"A checksum is a count of the number of bits in a transmission unit that is included with the unit so that the receiver can check to see whether the same number of bits arrived. If the counts match, it's assumed that the complete transmission was received." 

package com.javacodegeeks.java.core;
import java.io.FileInputStream;
import java.security.MessageDigest;

public class FileCheckSumExample {
    public static void main(String args[]) throws Exception {
        String filepath = "C:\\Users\\nikos7\\Desktop\\output.txt";
        MessageDigest messageDigest = MessageDigest.getInstance("SHA1");

        FileInputStream fileInput = new FileInputStream(filepath);
        byte[] dataBytes = new byte[1024];

        int bytesRead = 0;

        while ((bytesRead = fileInput.read(dataBytes)) != -1) {
            messageDigest.update(dataBytes, 0, bytesRead);
        }

        byte[] digestBytes = messageDigest.digest();

        StringBuffer sb = new StringBuffer("");
         
        for (int i = 0; i < digestBytes.length; i++) {
            sb.append(Integer.toString((digestBytes[i] & 0xff) + 0x100, 16).substring(1));
        }
        System.out.println("Checksum for the File: " + sb.toString());
        fileInput.close();
    }
}

If you run this, will generate a file's checksum value in Java using the SHA-1 hash function. Thus the basic steps are: (1) opening a MessageDigest instance, (2) open and reading the file using FileInputStream, (3) updating the MessageDigest with the new bytes read from the file using the MessageDigest.update method, and (4) using a StringBugger to convert and print the MessageDigest in hex representation.
