Finding GCD


Problem

Create a class that generates the Greatest common denominator between to numbers. Use Euclid’s algorithm.

Solution

The Euclid's algorithm is an efficient way to find GCD of two numbers and its pretty easy to implement using recursion in Java program. According to Euclid's method GCD of two numbers a, b is equal to GCD(b, a mod b) and GCD(a, 0) = a. The later case is a  based case for Java program to find GCD of two numbers using recursion

The Euclidean algorithm is based on the principle that the greatest common divisor of two numbers does not change if the larger number is replaced by its difference with the smaller number.


public int GCD(int a, int b){

   if (b==0) return a;

   return GCD(b,a%b);

}
