Basic Stats

Problem

Implement a Stats class that calculates the mean, std dev, and variance of an array of numbers.

Solution

Knowing the formulas for mean, standard deviation, and variance and knowing how to access the elements of an array is required to develop this solution. 

public class Stats{
	private double[] arr;

	public Stats(double[] array){
		arr = array;
	}

	public double mean(){
		if (arr.length == 0)
			return 0.0;
		double sum = 0;
		for(int i = 0; i < arr.length; i++)
			sum += arr[i];
		return sum / arr.length;
	}

	public double stdDev(){
		return Math.sqrt(variance());
	}

	public double variance(){
		if (arr.length == 0)
			return 0.0;
		double[] r = new double[arr.length];
		double mean = mean();
		for(int i = 0; i < arr.length; i++)
			r[i] = Math.pow((arr[i] - mean), 2);
		return new Stats(r).mean();
	}

}

If you run this, you need to instantiate the class with an array. When you call the methods mean, stdDev, and variance, you should get the mean of the array, the standard deviation of the array and the variance of the array.
