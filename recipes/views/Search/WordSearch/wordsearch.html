Prefix/Suffix Search

Problem

Given some text, a prefix/suffix flag, and a query word, find all prefix/suffix instances in the text. Sort by closeness.

Solution

Here is the code: 

  public class wordSearch{
  public static void wordSearch(String prefix, boolean isPrefix, String[] w) {
    if isPrefix {
      String[] wMatching;
      for (str : w) {
        if (str.regionMatches(true, 0, prefix, 0, prefix.length())) { // check if it's matching
          wMatching.append(str);
        }
      }
    } 
    else { 
      //reverse it
      for (str : w) {
        str = new StringBuilder(str).reverse().toString();
      }
      // isPrefix code which returns a wMatching list
      String[] wMatching;
      for (str : w) {
        if (str.regionMatches(true, 0, prefix, 0, prefix.length())) { // check if it's matching
          wMatching.append(str);
        }
      }
      // reverse it back
      for (str : wMatching) {
        str = new StringBuilder(str).reverse().toString();
      }
    }
    return wMatching;
  }
}

You'll want to use a ArrayList for your wMatching. I'm assuming you first wrote this in python? ArrayList is the closest thing to python's array. Java's Array [ ] is much more limited (you can only access and place elements in indices). Good strategy on the reversal. Just remember to reverse the prefix too! (e.g. prefix: go => og ... word: pogo => ogop)
