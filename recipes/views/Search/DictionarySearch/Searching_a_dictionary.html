Searching a Dictionary


Problem

Given an array of text and a query word, find all instances of the word and return the index of the array, and the position of the word in the text.


Solution

We import a txt file, and use the scanner method to turn it into an array of words. 

import Helper.txt;

Scanner sc = new Scanner(new File(Helper.txt));

List<String> lines = new ArrayList<String>();

while (sc.hasNextLine()) {

  lines.add(sc.nextLine());

}

String[] textarray = lines.toArray(new String[0]);

Then we define a function getAllIndexes to get the index of a word, i.e. find word “She”

public static Array getAllIndexes(arr, val) {

    var indexes = [], i = -1;

    while ((i = arr.indexOf(val, i+1)) != -1){

        indexes.push(i);

    }

    return indexes;

}


var indexes = getAllIndexes(textarray, "She");
