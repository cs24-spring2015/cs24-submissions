Batch File Opening

Problem

You want to see all the files of a certin type.

Solution 

// Case.java

import java.io.File;

public class ListFiles {
    
    public static void main(String[] args) {
        
        // Directory path here
        String path = args[0];
        String type = args[1];
        
        String files;
        File folder = new File(path);
        File[] listOfFiles = folder.listFiles();
        
        for (int i = 0; i < listOfFiles.length; i++)
        {
            
            if (listOfFiles[i].isFile())
            {
                files = listOfFiles[i].getName();
                if (files.endsWith(type))
                {
                    System.out.println(files);
                }
            }
        }
    }
}

If you run this, it prints the file names that are of the same file type specified:

ListFiles (DIRECTORY_LOCATION) (FILE_TYPE)
